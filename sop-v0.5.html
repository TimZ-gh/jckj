01

Operation / Listing

运营 / Listing SOP

目标：把“选品、扒品、定价、上架、出单后反馈”整理为可复用流程，降低人工重复上架和数据复制错误。

选品候选第三方平台、竞品、国家、销量
侵权 / 成本审核品牌、外观、采购价、重量体积
货源与售价核算1688 找同款、比价、利润测算
扒品图包图片、标题、详情页、规格
Listing 审核标题、关键词、五点描述、资质图
上架 / 活动链接、店铺、国家、价格、状态
出单 / 反馈发货、退款拦截、差评、数据复盘

输入资料

  • 选品表、竞品链接、第三方平台数据。
  • 1688 采购价、重量、体积、供应商信息。
  • 图包、标题、关键词、资质图、店铺后台规则。

人工把控

  • 是否侵权、是否可采购、是否有价格优势。
  • 标题 / 图包 / 详情页是否合规。
  • 上架失败、违规词、差评和退款处理。

AI 可辅助

  • 多平台选品数据汇总和竞品采集。
  • 1688 找货源、比价、利润测算。
  • 图包整理、标题生成、违规词初筛、批量上架辅助。

待追问

  • 选品表和上架表样例。
  • 侵权判断规则和违规词记录。
  • 多国家 / 多店铺上架差异。
02

Content / Video

内容 / 视频 SOP

目标：把 AI 图片、AI 视频、Prompt、剪辑审核和成片反馈变成可沉淀、可复用、可追踪的内容生产流程。

选题 / 参考抖音、TK、B 站、产品卖点
创意与脚本场景、角度、卖点、节日氛围
Prompt 生成系统提示词、用户提示词、反推提示词
AI 出图 / 出视频RunningHub、ComfyUI、即梦、小云雀
剪辑包装剪映、字幕、音频、特效、修复错误
审核确认产品、画面、重复率、品牌标识
成片归档素材、Prompt、成片、表现数据

输入资料

  • 产品图、卖点、参考视频、素材库。
  • Prompt 文档、系统提示词、用户提示词。
  • 手机视频内容分析表、达人转化表。

人工把控

  • 提示词是否准确，产品外观是否错误。
  • 画面是否错位、是否出现品牌标识。
  • 素材重复率、非原创风险、最终成片质量。

AI 可辅助

  • 图片反推、图生图、视频提示词扩写。
  • 脚本初稿、多版本素材生成、视频放大。
  • 成片初筛和素材分类辅助。

待追问

  • Prompt 文档样例和目录结构。
  • 成片审核评分标准。
  • 非原创 / 打回记录字段。
03

BD / Creator

BD / 达人 SOP

目标：把达人筛选、批量邀约、寄样判断、发布链接回收和达人分级变成标准流程，减少靠人工记忆推进。

达人筛选类目、粉丝、GPM、履约度
批量邀约邀请函、私信、商品卡链接
沟通判断申样、报价、是否愿发视频
寄样决策先发视频、出单后寄样、先寄样
寄样跟进快递单号、到货、履约时间
发布回收链接、播放、互动、转化
达人分级优质、可跟进、低价值、黑名单

输入资料

  • 达人表、达人主页、粉丝量、GPM、类目。
  • BD 话术、商品卡链接、合作记录。
  • 寄样记录、发布链接、达人视频数据。

人工把控

  • 是否寄样、寄哪个产品、是否长期维护。
  • 合作价格、达人优先级、赔偿承诺边界。
  • 达人是否适合当前产品和国家市场。

AI 可辅助

  • 达人筛选、标签归类、批量邀约。
  • 话术生成、跟进提醒、发布链接检测。
  • 达人数据导出、视频类型识别、表现汇总。

待追问

  • 达人表样例和字段含义。
  • 达人分级标准。
  • 成功 / 失败达人案例。
04

Warehouse / Procurement

仓储 / 采购 SOP

目标：统一多仓库存、采购、寄样、出库、物流异常和退货入库流程，解决多系统数据不共享和人工补标问题。

采购 / 入库供应商、采购价、批次、SKU
库存更新国内、欧洲、美国、英国、多系统
寄样申请BD、达人、SKU、数量、地址
核库存 / 打标样品订单、达人订单、商品卡订单
出库 / 物流物流单号、签收、异常
退货 / 异常缺货、错发、丢件、延迟
周转 / 补货库存价值、周转率、周转天数

输入资料

  • 库存表、进销存导出表、样品出库表。
  • 寄样申请表、物流问题表、供应商表。
  • 采购登记表、银行卡 / 支付宝流水。

人工把控

  • 国内库存盘点、样品订单打标。
  • 退货入库、物流异常、采购付款审核。
  • 缺货、错发、延迟、丢件处理方案。

AI 可辅助

  • 整合多系统进销存和库存周转分析。
  • 物流轨迹异常提醒、订单自动分类打标。
  • 采购数据、流水、付款登记自动核对。

待追问

  • 多仓库存表字段样例。
  • 寄样申请单和物流问题表。
  • 采购付款审核表。
05

Finance / Statistics

财务 / 统计 SOP

目标：把订单、采购价、物流费、回款、异常、付款申请和合作店铺利润统计统一到可核对、可追溯的财务台账流程。

订单导出美区、英区、欧区、全量订单
价格匹配订单编号、采购价、物流价
异常清单价格为空、费用异常、报价不符
付款 / 发票付款申请、物流发票、归档
回款台账已回款、未回款、部分回款
收支统计收入、退款、物流费、杂费
利润确认合作店铺营收、成本、净利、分成

输入资料

  • 订单明细、物流费表、采购价表。
  • 回款明细、收支明细、异常订单清单。
  • 合作店铺利润统计表。

人工把控

  • 付款申请、发票合规确认。
  • 收款账号申请与绑定。
  • 异常订单处理和利润口径确认。

AI 可辅助

  • 多文件合并、格式统一、订单编号匹配。
  • 自动标记异常订单并生成异常清单。
  • 自动归集收入支出，按店铺生成收支表。

待追问

  • 订单明细和物流费表样例。
  • 利润统计口径和分成规则。
  • 异常订单分类标准。
